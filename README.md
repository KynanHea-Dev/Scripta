# Scripta
A beautiful Bible reading tracker inspired by Apple's Liquid Glass design. Track chapters, monitor progress, take notes, build reading streaks, and customize your experience — all stored locally on your device.

# features
## Features

* 📖 Track all 66 books and 1,189 chapters of the Bible
* ✅ Automatic book completion
* 📝 Personal notes for every book
* 📊 Reading statistics and progress tracking
* 🔥 Daily reading streaks
* 📚 Recent reading history
* 🔍 Instant book search
* 🎨 Multiple themes and custom colors
* 📂 Expand and collapse books
* 💾 Local storage with no account required
* 📤 Export and import backups
* 📱 Mobile-friendly responsive design
* ✨ Apple-inspired Liquid Glass interface
* 🔒 Private and offline-friendly
